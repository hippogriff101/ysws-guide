# ysws-guide

look at that - it's a website to teach people how to make a YSWS - thats so cool!

 If you are not familiar YSWS stands for You Ship We Ship. Hack Club runs events regularly where you ship (publish a project) and then Hack Club will send you something in return. YSWS at Hack Club are sponsored by a member of the Hack Club staff, most likely a Gap Year / Intern.
